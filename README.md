# Jarvis OS // Motor Financeiro Pessoal 🤖💰

Assistente pessoal inteligente com estética **Noir Premium**, focado em controle financeiro via WhatsApp, totalmente integrado ao Firebase.

## ✨ Visual "Noir Premium" (100% em Português)
O sistema utiliza uma estética inspirada em interfaces de hardware de alto desempenho e design de luxo:
- **Painéis Monofont**: Dados financeiros formatados para máxima precisão.
- **Acentos Cyber-Cyan**: Detalhes técnicos e indicadores de status em azul ciano.
- **Glassmorphism**: Elementos de interface semi-transparentes com desfoque de fundo.
- **Navegação Fluida**: Abas funcionais para Dashboard, Relatórios e Configurações.

## 🚀 Funcionalidades
- **Cérebro Multimodal**: IA (OpenRouter/Gemini) interpreta seus gastos naturais em português.
- **Firebase Core**: Armazenamento em tempo real usando Cloud Firestore.
- **WebSocket Sync**: Sincronização instantânea entre o WhatsApp e o Dashboard.
- **Diretrizes Neurais**: Lembretes e metas de economia gerenciados via chat.

## ☁️ Hospedagem
O projeto está configurado para o ambiente Cloud Run com persistência no Firebase Firestore.

## 🛠️ Configuração Rápida
1. Adicione sua `OPENROUTER_API_KEY` ao `.env`.
2. O Firebase foi configurado automaticamente pelo sistema.
3. `npm install`
4. `npm run dev`
5. Escaneie o QR Code no dashboard para ativar o Jarvis.
